# css文本的属性

### text-decoration

给文本添加装饰线如下划线、删除线

### text-transform

用于设置文字字母大小写的转换

- capitalize：将每个单词的首字母大写
- uppercase：将所有字母大写
- lowercase：将所有字母小写
- none：没有任何影响

### text-indent

设置第一行内容的缩进



### text-align(重要)

定义***行内元素***内容（例如文字）如何相对它 的块父元素对齐；

- left：左对齐
- right：右对齐
- center： 正中间显示
- justify：两端对齐

### word/letter-spacing

letter-spacing、word-spacing分别用于设置字母、单词之间的间距：默认是0，可以设置为负数
